int interactSpaceObject(SpaceObject* spaceObject, SpaceObject* ship)

Facilitates colonization of a planet. There are three scenarios that can occur while a spaceship is exploring: 1) spaceship encounters empty planet: immediately colonizes. 2) spaceship encounters occupied enemy planet: battle occurs between the two aliens; if the spaceship wins, it colonizes the planet, otherwise the occupants remain on the planet and the spaceship explodes. 3) spaceship encounters friendly planet or star: no action occurs.

Parameters
spaceObject A SpaceObject, which can be a planet or star.
ship The ship that is encountering the SpaceObject.
Returns int 0 = no action, 1 = planet occupants win, 2 = ship colonizes.
